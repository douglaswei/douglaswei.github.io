hadoop-入门

Douglas 2016-04-13 13:05:43
大数据

CDH安装配置

安装

配置

hadoop生态圈组件

Apache Hadoop（Common、HDFS、MapReduce、YARN）

数据操作系统，在hdfs之上提供了对数据资源的调度

Apache HBase

hdfs之上的Nosql数据库；
主要概念有：行键、列簇、列修饰符、版本号

Apache ZooKeeper

分布式资源服务的协调平台

Apache Oozie

工作流引擎服务器，基于MR定义了工作流

Apache Hive

基于hadoop的数据仓库，可以将结构化的数据文件映射成数据库表，并提供简单的查询功能，底层转化为MR

Hue（已获 Apache 许可）

hadoop的可视化工具

Apache Flume

数据流中间件；类似的产品为logstash

Cloudera Impala（许可的 Apache）

类似于hive，区别是不急于MR，更快速

Apache Sentry

python的日志分析平台

Apache Sqoop

ETL工具，用于在hive和传统sql数据库之间传输数据

Cloudera Search（许可的 Apache）

solr

Apache Spark(todo)

Apache Storm(todo)
